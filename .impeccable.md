## Design Context

### Users
Multi-role system for college/event management:
- **College administrators**: Full access to manage events, participants, colleges, and rankings
- **Event coordinators**: Track events and adjust points
- **Students/participants**: View leaderboards and results

### Brand Personality
**Clean & Professional** with a distinctive edge:
- Sophisticated dark theme (Wizardly Obsidian) 
- Teal accent color for vibrancy
- Not generic - avoid AI slop aesthetics
- Trustworthy, straightforward, data-focused

### Aesthetic Direction
- Dark mode primary (pure-black premium feel)
- Teal accents (#7cebd6) for interactive elements
- Clean typography (Sora + JetBrains Mono)
- Information-dense but organized
- No excessive decoration, glassmorphism, or gradient text

### Design Principles
1. **Clarity over flash** - Every element serves a purpose
2. **Professional authority** - Trustworthy, not playful or childish
3. **Efficient information** - Dense but scannable layouts
4. **Distinctive, not derivative** - No Inter/Roboto, no cyan-on-dark clichés
5. **Dark refined** - Premium dark mode, not "gamer" aesthetic

### Technical Stack
- React + TypeScript + Vite
- Tailwind CSS + shadcn-ui
- Dark mode by default
